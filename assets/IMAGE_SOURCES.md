## 🎯 Strategi Download & Penggunaan

### Untuk Products (12+ gambar):
```
1. Cari di Unsplash: "modern sofa"
2. Pilih gambar dengan background clean/white
3. Download high quality (1920px+)
4. Crop square (1:1 ratio) menggunakan image editor
5. Compress dengan TinyPNG
6. Rename: sofa-modern-01.jpg
7. Upload ke: assets/images/products/
```

### Untuk Categories (6 gambar):
```
1. Cari di Pexels: "living room interior"
2. Download 4:3 ratio image
3. Resize ke 1200x900px
4. Compress dengan Squoosh
5. Rename: living-room.jpg
6. Upload ke: assets/images/categories/
```

### Untuk Hero Banner (1-3 gambar):
```
1. Cari di Unsplash: "modern interior living room wide"
2. Download high quality landscape
3. Resize ke 1920x1080px (Full HD)
4. Compress dengan TinyPNG
5. Rename: hero-banner-01.jpg
6. Upload ke: assets/images/hero/
```

## ✅ Checklist Before Upload

- [ ] Image sudah di-compress (< 200KB untuk products)
- [ ] Format sesuai (JPG untuk foto, PNG untuk transparency, WebP untuk modern browsers)
- [ ] Size/resolution sesuai kebutuhan
- [ ] Naming convention konsisten (lowercase, dash-separated)
- [ ] Quality masih bagus setelah compress
- [ ] Punya license untuk penggunaan komersial

## 💡 Pro Tips

1. **Konsistensi Style**: Pilih gambar dengan style yang konsisten (modern, minimalist, dll)
2. **Background**: Untuk products, pilih background putih/neutral
3. **Lighting**: Konsisten lighting untuk semua product images
4. **Angle**: Gunakan angle yang sama untuk produk sejenis
5. **Color Palette**: Sesuaikan dengan color scheme website (#D4AF37)
6. **Resolution**: Lebih besar lebih baik (bisa resize down, tapi tidak bisa up)
7. **Watermark**: Hindari gambar dengan watermark

## 🚀 Quick Start

**Butuh gambar cepat untuk testing?**

1. Buka: https://unsplash.com/s/photos/furniture
2. Download 12 gambar furniture berbeda
3. Buka: https://tinypng.com/
4. Upload & compress semua gambar
5. Rename sesuai convention
